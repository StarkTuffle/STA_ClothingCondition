# StarkTruffle Additions - Clothing Condition

> A clothing condition randomizer mod for Project Zomboid

---

## Overview

This will randomly deplete the condition of clothing found throughout the Knox County. Depletion chance and levels can be configured through sandbox settings.

---

## Features

- Clothing stores can be configured to have better quality.
- Works with modded clothing and the Item Stories mod.
- Customizable with sandbox settings.
